The WMT Execution Server
==================================================================

:Release: |release|
:Date: |today|

The WMT Execution Server,
part of the `CSDMS Web Modeling Tool <http://csdms.colorado.edu/wiki/WMT_portal/>`_,
provides tools to configure and run stand-alone or coupled models
on a computational resource.
The source for this package can be found on GitHub at
`https://github.com/csdms/wmt-exe <https://github.com/csdms/wmt-exe/>`_.
